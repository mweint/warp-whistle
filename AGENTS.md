# SMB3 Editor project instructions

For every design, implementation, review, or test change in this repository, read and follow [skills/smb3-vanilla-guardrails/SKILL.md](skills/smb3-vanilla-guardrails/SKILL.md).

Vanilla compatibility is the default. Do not change fundamental SMB3 mechanics, executable behavior, mapper, or ROM dimensions unless the user explicitly requests a chocolate feature and the work is isolated behind a deliberate mode boundary.

Keep all user-facing commentary and final responses concise and token-efficient. Report only the outcome, important caveats, tests, and runnable artifact path unless the user asks for detail. Avoid repeating prior context, narrating routine steps, or emitting large logs.
