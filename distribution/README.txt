Warp Whistle portable workspace

1. Put a verified SMB3 PRG0 or PRG1 ROM in ROMs.
2. Put an emulator such as Mesen in Emulators.
3. Start WarpWhistle.exe.

Projects stores editor project files. Exports is a convenient place for ROMs and
patches made by the editor. Data stores your editor settings, palette library,
and temporary play-test ROMs. No ROM-derived item-image cache is written to disk.
